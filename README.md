# Kubeflow---Natural-Language-Processing
## Version
* Docker：20.10.7
* Kubectl：v1.20.1
* Minikube：v1.20.0
* Kubeflow：v1.3

If you install kubeflow v1.3, you can skip the install.

## Install

    sudo chmod +x docker.sh
    sudo chmod +x minikube.sh
    sudo chmod +x kubeflow.sh
or
    sudo chmod +x *
